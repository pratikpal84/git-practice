# Kana Practice

A simple web application for practicing Japanese Hiragana and Katakana characters.

## Features

* Practice Hiragana and Katakana
* Score and accuracy tracking
* Streak counter
* Dark mode support
* Japanese pronunciation using speech synthesis
* Mobile-friendly design

## Technologies Used

* HTML
* CSS
* JavaScript
